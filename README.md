# Scraper

## Introduction

You have to change databasename, user_name and password before using.
## Usage

Clone the project

```bash
  https://github.com/grealyve/news_scraper.git
```

Install required library

```bash
  pip install selenium
```
```bash
  pip install pandas
```

Run the python code

```bash
  python scraper.py
```

  
## Used Technologies

Python3, Selenium, Pandas
